# MemoryWall Instruction Manual

This file is kept as a top-level quick manual. For the fuller Version 1 documentation set, see [docs/INDEX.md](/D:/memorywall/docs/INDEX.md).

## Quick Start

1. Open the app.
2. Go to `Collect`.
3. Upload photos/videos.
4. Optionally upload a main audio file.
5. Go to `Arrange`.
6. Reorder items and edit labels if needed.
7. Go to `Share`.
8. Preview, export, or share.

## Main Sections

### Collect

- Upload photos and videos
- Upload optional background audio
- Review small file tiles

### Arrange

- Drag cards to reorder
- Edit file name, event name, and tag

### Share

- View A4-style preview
- Play slideshow
- Export PDF/PNG
- Share PDF through native share/WhatsApp fallback

## Important Notes

- Leaving `event name` blank keeps it hidden
- Leaving `tag` blank keeps it hidden
- PDFs do not include music playback
- V1 does not yet generate a permanent public internet link
